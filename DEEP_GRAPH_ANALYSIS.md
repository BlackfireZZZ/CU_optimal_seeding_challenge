## 8. Cascade Dynamics - Why Single Seeds Usually Fail

Mean degree = 42.5. Average node needs ceil(0.18 * 42.5) = 8 infected neighbors.
A single seed is only 1 infected neighbor for each direct neighbor.

| Node degree | Infected neighbors needed | Count in graph |
|-------------|--------------------------|----------------|
| 1-5 | 1 | 482 nodes |
| 6-10 | 2 | ~400 nodes |
| 11-20 | 2-4 | ~500 nodes |
| 21-50 | 4-9 | ~1,100 nodes |
| 51+ | 10+ | ~1,470 nodes |

482 nodes need only 1 infected neighbor — easiest cascade targets.
~2,800 nodes need 8+ neighbors — require coordinated multi-seed attack.

**Why magic seeds work despite this:**
1. Their few low-degree neighbors cascade immediately (only need 1-2 infected friends)
2. Triggered neighbors push their mutual friends over threshold
3. Chain reaction amplifies through community echo-chamber structure
4. Specific topology creates avalanche in the community

## 14. Agent Implementation Guide

### Optimal strategy pseudocode

```python
PRIORITY_SEEDS = [
    # Phase 1 - Day 0 (within 10,000 rub budget)
    3057,  # deg=16, 4800 rub, 700 viral -> 30,200 rub profit
    3775,  # deg=8,  2400 rub, 492 marginal viral
    2263,  # deg=6,  1800 rub, 226 marginal viral
    3991,  # deg=2,  600 rub,  44 marginal viral
    443,   # deg=1,  300 rub,  16 marginal viral
    # Phase 2 - Days 1-6 (early income)
    2788,  # 300 rub,  +350 rub marginal profit
    454,   # 600 rub,  +600 rub marginal profit
    167,   # 1800 rub, +1600 rub marginal profit
    93,    # 2100 rub, +1300 rub marginal profit
    337,   # 2400 rub, +1000 rub marginal profit
    # Phase 3 - Day ~11 (need 11,100 rub accumulated)
    1304,  # 11100 rub, +24,400 rub marginal profit
]

active = set()
budget = 10000
queue = list(PRIORITY_SEEDS)

for day in range(60):
    seeds_today, next_queue = [], []
    for v in queue:
        if v not in active and 300*degree[v] <= budget and len(seeds_today) < 10:
            seeds_today.append(v); budget -= 300*degree[v]
        else:
            next_queue.append(v)
    queue = next_queue
    active |= set(seeds_today)

    new_viral = spread_one_step(active)  # LT model threshold=0.18
    budget += len(new_viral) * 50
    active |= new_viral
```
